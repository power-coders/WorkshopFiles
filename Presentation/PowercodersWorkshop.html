Web Development

Web Devolpment has two parts:

  • Frontend: How the Website looks and behaves.
  • Backend: How to save, process and deliever data.

Frontend vs. Backend

In Frontend Development there are three parts you need to master.

  • HTML: Defines the content
  • CSS: Styles the content
  • Javascript: Interactivity of the content

Backend is much more diverse. It includes Databases, always programming, servers and so on. We will not focus on it today.

HTML

HTML is not a programming language but a Markup Language. In a Markup Language you can define the different elements with tags. This way our browser can differentiate between text and other elements.

HTML

<h2>HTML</h2>
As you can see you need a opening tag then you define the content and then the closing tag 
<a href="www.duck.com">DuckDuckGo</a>
We can define additional information in the opening tag. In this case we defined a hyperlink.

CSS

CSS is short for Cascading Style Sheet. With CSS you can style every element as you wish. Most essential you can change size, color and position of elements.

CSS

Here you see a css example: 

p {
  font-size: 1.9 rem;
  color: red;
}

There are some ways to select the specific element you want to theme. The example above will theme all content inside a p-tag.

Javascript

Javascript is a high-level programming language. You can do anything with Javascript. Javascript is giving you a lot of option to interact with the HTML elements. Most importantly you can change style and content of elements. This makes it possible to add interactivity to a website.

Programming

  • Programming is the art of breaking a problem down into smaller problems.
  • Finding the solutions to these smaller problems.
  • Putting these pieces back together.

Building blocks of Programming:

  • Variables
  • Conditions
  • Loops
  • Functions

Variables

Variables are the way we store and access data in programming. We can store simple things like numbers, strings (text) but also mor complex like objects and arrays.

let Programming = 'cool';
let numberOfHouses = 542;
let numberOfPersonsLivingInHouses = 2.5 * numberOfHouses;

Conditions

Conditions is the way we steer the program.
A condition is for example if the weather is nice then go outside else stay inside.

if(weather == 'nice'){
    goOutside();
} else {
  stayInside();
}

Loops

With Loops our programms get really powerful, It is a way to run lines of code as often as you want. An example would be: As long as the music is playing draw circles. We have a condition There are a few loop options available. We will have a look at the most used one the for loop.

For-Loop

The For-Loop has three part, the initialization which is only done one time ate the beginning, then the condition which determines when to stop and then there is a command that gets executed every loop we make. 

for(let index = 0; index < 100; index = index + 1){
    drawCircle();
}
